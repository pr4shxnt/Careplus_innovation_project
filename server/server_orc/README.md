# Bun Backend Template

A minimal default backend template built with Bun.

## Features

- `GET /` service status payload
- `GET /health` health check endpoint
- Consistent JSON responses
- Port resolution with `PORT` env support
- Bun test coverage for route behavior

## Install

```bash
bun install
```

## Run

```bash
bun run dev
```

or

```bash
bun run start
```

## Test

```bash
bun run test
```

## Endpoints

- `GET /`
- `GET /health`
