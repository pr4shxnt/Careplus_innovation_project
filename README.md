# Innovation_Project

Innovation project for semester 2 in Sunway College Kathmandu

## Idea 1: Carebot for the people suffering from dementia

Type: Multi-platform (Application + IoT)

Application development:

- React-Native (Expo)
- Node.js (Express)
- MongoDB

IoT development:

- Raspberry Pi
- Microphone module
- Speaker module

AI development:

- Local LLM: Gemma
- RAG pipleline

#### Features:

    - AI based loneliness companion
    - Mood tracking
    - Agentic medicine tracking and objects tracting
    - Agentic reminders
    - Agentic medicine and registration
    - Notification of medicine reminders on application
    - Voice based interaction with the carebot
    - TTS and STT for voice based interaction
